### 설치 및 시작하는 법

- `git clone https://github.com/Derek-94/hayanmind.git`
  </br>
- `yarn install`
  </br>
- `yarn start`

### 프로젝트 구조 설명

- `Container` 에서 렌더링에 필요한 데이터를 `fetch` 한다.
  </br>
- `Presenter` 는 `props` 로 `fetch` 결과물을 받아 렌더링을 한다.

  </br>
